# Keuzehulp Verduurzaming — Parallelweg 52

Interactieve keuzehulp voor het verduurzamingsproject van onze woning.

## Over dit project
- 2-onder-1-kap, bouwjaar ~1935, Koog aan de Zaan
- Energielabel F → doel: label A
- Alle maatregelen, kosten, subsidies en opbrengsten op één plek

## Gebruik
Open de live URL op je telefoon of laptop. Alle keuzes worden opgeslagen in je browser.

## Wijzigingen doorvoeren
Geef de gewenste aanpassing door in Claude, kopieer de nieuwe `index.html` en push naar GitHub. Vercel deployt automatisch.
